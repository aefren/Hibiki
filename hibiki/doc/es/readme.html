Hibiki - Navegación con Sonidos Espaciales

Versión: 0.8.1

Autor: Alfred

Descripción

Hibiki es un complemento (add-on) para NVDA que proporciona retroalimentación auditiva mediante sonidos espaciales 3D para diferentes tipos de controles. En lugar de depender únicamente de anuncios verbales repetitivos, cada tipo de control (botones, enlaces, casillas de verificación, etc.) reproduce un sonido distintivo que ayuda a identificarlo de manera más rápida e intuitiva.

Características Principales

  • Audio Espacial 3D: Los sonidos se posicionan en el espacio según la ubicación del control en la pantalla, proporcionando información sobre su posición (izquierda, derecha, centro).
  • Sonidos Distintivos: Más de 25 tipos de controles diferentes tienen sonidos únicos (botones, enlaces, casillas, menús, campos de texto, etc.).
  • Supresión de Etiquetas de Rol: Opción configurable para eliminar los anuncios verbales de roles ("botón", "enlace", etc.) y depender únicamente de sonidos.
  • Retroalimentación de Estados: Sonidos adicionales para estados como "marcado", "expandido" y "contraído".
  • Fácil Activación/Desactivación: Alterna el complemento con un simple atajo de teclado.

Instalación

  1. Descarga el archivo Hibiki-0.8.1.nvda-addon
  2. Abre el archivo (se abrirá con NVDA)
  3. Sigue las instrucciones para instalar el complemento
  4. Reinicia NVDA cuando se te solicite

Uso

Navegación Básica

Una vez instalado, Hibiki funcionará automáticamente mientras navegas con NVDA:

  • Navegación con Teclado: Al presionar Tab, Shift+Tab o las teclas de flecha, escucharás sonidos distintivos para cada control al que te muevas.
  • Navegación de Objetos NVDA: Al usar los comandos de navegación de objetos (NVDA + flechas del teclado numérico), también escucharás sonidos.

Atajos de Teclado

Atajo Descripción
NVDA+Shift+S Activa o desactiva Hibiki

Configuración

Para acceder a las opciones de configuración:

  1. Abre el menú de NVDA (NVDA+N)
  2. Selecciona "Preferencias" > "Opciones..."
  3. En el cuadro de diálogo de Opciones, busca la categoría "Hibiki"

Opciones Disponibles

  • Habilitar Hibiki: Activa o desactiva el complemento completamente.
  • Suprimir etiquetas de rol habladas: Cuando está activada, NVDA no dirá las etiquetas de rol como "botón" o "enlace", reproduciendo únicamente el sonido. Cuando está desactivada, se escucharán tanto los sonidos como las etiquetas de rol.

Ejemplos de Sonidos

Cada tipo de control tiene un sonido característico:

  • Botones: Sonido de clic suave
  • Enlaces: Sonido de agua burbujeante
  • Casillas de verificación: Sonido de interruptor
  • Campos de texto: Sonido de tecleo
  • Menús: Sonido de campanilla
  • Pestañas: Sonido de papel pasando
  • Y muchos más...

Requisitos del Sistema

  • NVDA 2019.3 o posterior
  • Windows 7 o posterior
  • Tarjeta de sonido compatible con audio estéreo

Compatibilidad

Hibiki ha sido probado con:

  • NVDA 2019.3 hasta 2025.3
  • Navegadores web: Firefox, Chrome, Edge
  • Aplicaciones de Windows: Explorador de archivos, Microsoft Office, Configuración

Solución de Problemas

No escucho ningún sonido

  • Verifica que Hibiki esté habilitado (NVDA+Shift+S)
  • Comprueba la configuración en Preferencias > Opciones > Hibiki
  • Asegúrate de que el volumen del sistema no esté silenciado

Los sonidos interfieren con el sintetizador de voz

  • Puedes ajustar el volumen del sintetizador en la configuración de NVDA
  • Considera habilitar la opción "Suprimir etiquetas de rol habladas" para reducir la redundancia

NVDA se comporta de manera extraña después de instalar

  • Intenta desactivar el complemento temporalmente (NVDA+Shift+S)
  • Si el problema persiste, desinstala el complemento y reinicia NVDA
  • Reporta el problema al equipo de desarrollo

Créditos y Agradecimientos

Hibiki está inspirado en el complemento Unspoken de Austin Hicks, que fue pionero en el uso de sonidos espaciales 3D para la navegación con NVDA. Agradecemos su trabajo innovador y su contribución a la comunidad de usuarios de lectores de pantalla.

Este complemento utiliza:

  • camlorn_audio - Biblioteca de audio 3D
  • OpenAL Soft - Motor de audio espacial
  • libsndfile - Biblioteca de carga de archivos de audio
  • Sonidos del complemento Unspoken original
