Hibiki - Spatial Audio Navigation

Version: 0.8.1

Author: Alfred

Description

Hibiki is an NVDA add-on that provides audio feedback through 3D spatial sounds for different control types. Instead of relying solely on repetitive verbal announcements, each control type (buttons, links, checkboxes, etc.) plays a distinctive sound that helps identify it more quickly and intuitively.

Key Features

  • 3D Spatial Audio: Sounds are positioned in space based on the control's location on the screen, providing information about its position (left, right, center).
  • Distinctive Sounds: Over 40 different control types have unique sounds (buttons, links, checkboxes, menus, text fields, etc.).
  • Role Label Suppression: Configurable option to suppress verbal role announcements ("button", "link", etc.) and rely solely on sounds.
  • State Feedback: Additional sounds for states like "checked", "expanded", and "collapsed".
  • Easy Toggle: Toggle the add-on with a simple keyboard shortcut.
  • Custom Sounds: Assign custom WAV files to any control type.
  • Browse Mode Support: 3D sounds during arrow key navigation in web browsers.

Installation

  1. Download the Hibiki-0.8.1.nvda-addon file
  2. Open the file (it will open with NVDA)
  3. Follow the instructions to install the add-on
  4. Restart NVDA when prompted

Usage

Basic Navigation

Once installed, Hibiki will work automatically as you navigate with NVDA:

  • Keyboard Navigation: When pressing Tab, Shift+Tab, or arrow keys, you will hear distinctive sounds for each control you move to.
  • NVDA Object Navigation: When using NVDA's object navigation commands (NVDA + numpad arrows), you will also hear sounds.
  • Browse Mode: When navigating web pages with arrow keys or quick navigation keys (k, b, e, h, etc.), spatial sounds play for each element.

Keyboard Shortcuts

Shortcut Description
NVDA+Shift+S Toggle Hibiki on/off

Configuration

To access configuration options:

  1. Open the NVDA menu (NVDA+N)
  2. Select "Preferences" > "Settings..."
  3. In the Settings dialog, find the "Hibiki" category

Available Options

  • Enable Hibiki: Turns the add-on completely on or off.
  • Suppress spoken role labels: When enabled, NVDA will not say role labels like "button" or "link", playing only the sound. When disabled, both sounds and role labels will be heard.
  • Suppress spoken state labels: When enabled, NVDA will not say state labels like "checked" or "expanded".
  • Enable browse mode sounds: Play spatial sounds during arrow key navigation in web browsers and other documents.
  • Customize Sounds: Assign custom WAV files to specific control types.

Sound Examples

Each control type has a characteristic sound:

  • Buttons: Soft click sound
  • Links: Bubbling water sound
  • Checkboxes: Switch sound
  • Text Fields: Typing sound
  • Menus: Chime sound
  • Tabs: Paper rustling sound
  • Headings: Heading notification sound
  • Tables: Table notification sound
  • And many more...

System Requirements

  • NVDA 2019.3 or later
  • Windows 7 or later
  • Sound card compatible with stereo audio

Compatibility

Hibiki has been tested with:

  • NVDA 2019.3 through 2025.3
  • Web browsers: Firefox, Chrome, Edge
  • Windows applications: File Explorer, Microsoft Office, Settings

Troubleshooting

I don't hear any sounds

  • Verify that Hibiki is enabled (NVDA+Shift+S)
  • Check the configuration in Preferences > Settings > Hibiki
  • Make sure the system volume is not muted

Sounds interfere with the speech synthesizer

  • You can adjust the synthesizer volume in NVDA settings
  • Consider enabling "Suppress spoken role labels" to reduce redundancy

NVDA behaves strangely after installation

  • Try temporarily disabling the add-on (NVDA+Shift+S)
  • If the problem persists, uninstall the add-on and restart NVDA
  • Report the issue to the development team
